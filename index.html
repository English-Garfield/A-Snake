A* Pathfinding Algorithm

The AI uses the A* algorithm to find the optimal path to food while avoiding its own body. A* evaluates each cell using f(n) = g(n) + h(n), where g(n) is the cost from start to current node and h(n) is the heuristic distance to the goal. This ensures the shortest path is found efficiently.
